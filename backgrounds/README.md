# 背景图片使用说明

## 支持的图片格式
- PNG (*.png)
- JPEG (*.jpg, *.jpeg)
- BMP (*.bmp)
- TGA (*.tga)

## 推荐尺寸
- 最佳分辨率：800x600 或 1920x1080
- 宽高比：4:3 或 16:9

## 使用步骤
1. 将图片文件复制到这个文件夹中
2. 支持的文件名示例：
   - background.png
   - background.jpg
   - glass_bg.png
   - scene.jpg

## 注意事项
- 建议使用高质量的图片以获得最佳视觉效果
- 避免使用过大尺寸的图片，以免影响性能
- 透明背景的图片（PNG）也能正常工作

## 当前文件夹内容
此文件夹用于存放液态玻璃效果的背景图片。
